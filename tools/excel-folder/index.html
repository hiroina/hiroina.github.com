Excel Folder Add-In

2013-02-09

Excel Folder helps you manage folders.

It is simple - right click and select “Open Folder”. Then it creates folder ‘.\ Sheet Name \ “Cell A - Cell B” and open it.

Example

Put “2.1” in A10 cell, and “Elementary” in B10. And right-click and select “Open Folder”.

It makes Explorer creates and opens “.\Demo"2.1 - Elementary” folder.
